# Strokes

Strokes is a social media app developed using the MERN stack (MongoDB, Express.js, React, Node.js) technology. It leverages Tailwind CSS for the user interface, Cloudinary for storing post images, videos, and GIF files. The application includes features such as user registration, login, forgot password, post management, comment and like management, friend management, and account management.

## Project Domain

[Strokes - Visit the Website](https://stroke.vercel.app/)

## Technologies Used

- **MERN Stack:**
  - MongoDB: Database for storing application data.
  - Express.js: Backend framework for building robust APIs.
  - React: Frontend library for building user interfaces.
  - Node.js: JavaScript runtime for server-side development.

- **Tailwind CSS:**
  - A utility-first CSS framework used for styling the user interface.

- **Cloudinary:**
  - Cloud-based service for managing and delivering images and videos in your applications.

## Features

1. **User Authentication:**
   - Register and login securely.

2. **Post Management:**
   - Create, edit, and delete posts.

3. **Comment and Like Management:**
   - Add, manage, and like comments on posts.

4. **Friend Management:**
   - Connect with friends and manage your friend list.

5. **Account Management:**
   - Update account details and preferences.
